# Přispívání

Děkujeme vašemu zájmu o přispívání do tohoto projektu!

## Jak přispět

1. **Forkněte repository** - Vytvořte svůj vlastní fork projektu
2. **Vytvořte feature branch** - `git checkout -b feature/amazing-feature`
3. **Proveďte změny** - Upravte kód podle potřeby
4. **Commitujte změny** - `git commit -m 'Add amazing feature'`
5. **Pushnout na branch** - `git push origin feature/amazing-feature`
6. **Otevřete Pull Request** - Popište své změny a čekejte na review

## Pokyny pro kód

- Dodržujte Python PEP 8 coding standard
- Přidejte komentáře k složitějšímu kódu
- Aktualizujte dokumentaci pro nové funkce
- Testujte svůj kód na vzorových datech

## Hlášení chyb

Pokud najdete chybu:
1. Zkontrolujte, zda problém již není nahlášen v Issues
2. Pokud ne, vytvořte nový Issue s:
   - Jasným popisem chyby
   - Kroky reprodukce
   - Očekávaným chováním
   - Vaším prostředím (Python verze, OS, atd.)

## Nápady a návrhy

Máte ideup na vylepšení? Otevřete diskusi v Issues s tagem `enhancement`.

Děkujeme za vaše příspěvky! 🙏
